Features

Create

Intuitive pattern creation using multiple tiling types, including irregular polygon tessellation.

Import

Imports scanned photos, drawings or digital files from other software, including vector and raster file formats.

Save

Saves megapixel files up to 10,000 x 10,000 pixels.

Customise

Customises the stroke and fill parameters of your vector images.

Multiple Variations

Saves multiple variations, allowing for maximum creative flexibility.

A Flexible Tool

A great tool for anyone working with patterns: 3D artists, industrial designers, interior designers, graphic designers, fashion design, etc.
